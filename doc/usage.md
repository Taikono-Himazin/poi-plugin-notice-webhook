# 使い方

## 直接配信モード

poi が動作しているマシンから Webhook を直接送信します。クラウドの設定は不要です。

### 設定手順

1. poi の設定画面でプラグインを開く
2. **配信方法** で「直接配信」を選択
3. **Webhook の種類** を選択（Discord / Slack / 汎用）
4. **Webhook URL** を入力
5. 「テスト通知」ボタンで動作確認
6. 「保存」ボタンで保存

### Webhook URL の取得方法

**Discord**

1. 通知を送りたいチャンネルの設定を開く
2. 「連携サービス」→「ウェブフック」→「新しいウェブフック」
3. 作成したウェブフックの URL をコピー
   - 形式: `https://discord.com/api/webhooks/...`

**Slack**

1. [Slack App を作成](https://api.slack.com/apps) し、Incoming Webhooks を有効化
2. チャンネルを選択して Webhook URL を取得
   - 形式: `https://hooks.slack.com/services/...`

**汎用 Webhook**

任意の HTTP エンドポイントを指定できます。以下の JSON が POST されます。

```json
{
  "message": "遠征完了: 第1艦隊",
  "title": "遠征完了",
  "type": "expedition"
}
```

`type` の値: `expedition`（遠征）/ `repair`（入渠）/ `construction`（建造）/ `default`

## クラウド配信モード (PREMIUM)

AWS 経由でクラウドから通知します。[セットアップ](setup.md) が必要です。

### ログイン

1. **配信方法** で「クラウド経由」を選択
2. 「ログイン」ボタンをクリック
3. メールアドレス・パスワードでアカウントを作成またはサインイン
   （Google アカウントでのサインインが有効な場合は Google ボタンも表示されます）

### Webhook の設定

ログイン後、Webhook の種類と URL を設定して保存します。この設定はクラウドに保存され、通知はクラウドから送信されます。

### 通知タイミングの調整

クラウドモードでは完了の **N 分前** に通知を受け取れます（例: 5 分前に通知して余裕を持って帰投）。

設定画面の「完了前通知（分）」スライダーで調整してください。0 に設定すると完了時に通知されます。

### 有料プランへのアップグレード

無料プランは月 30 通まで。それ以上使用するには有料プランが必要です。

1. 設定画面の「お支払い」セクションを開く
2. プランを選択（1ヶ月 / 6ヶ月 / 12ヶ月）
3. 「購入する」ボタンをクリック
4. PAY.JP の決済ページでカード情報を入力
5. 決済完了後、自動的に有効化されます

有効期限が残っている状態で再購入すると、残り期間から延長されます。

### ログアウト

「ログアウト」ボタンをクリックするとアカウントからサインアウトします。ログアウト時にサーバー側でスケジュール済みの通知はキャンセルされます。

## 通知の種類とカラー

| 種類 | イベント | Discord カラー | Slack カラー |
|---|---|---|---|
| `expedition` | 遠征完了 | 青紫 `#5865F2` | 青紫 `#5865F2` |
| `repair` | 入渠完了 | 緑 `#57F287` | 緑 `#57F287` |
| `construction` | 建造完了 | 黄 `#FEE75C` | 黄 `#FEE75C` |
| `default` | その他 | グレー `#AAAAAA` | グレー `#AAAAAA` |

## テスト通知

設定画面の「テスト通知」ボタンで動作確認できます。実際のゲームイベントを待たずに Webhook の接続を確認するのに便利です。
