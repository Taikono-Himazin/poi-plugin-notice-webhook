# プライバシーポリシー

本プライバシーポリシーは、poi-plugin-notice-webhook（以下「本プラグイン」）の利用にあたって収集・利用する情報について説明します。

---

## 直接配送モードについて

直接配送モードでは、ゲームイベントの通知をお使いのマシンから直接 Webhook URL へ送信します。本プラグインがお客様のデータを外部サーバーへ送信・保存することはありません。

---

## クラウド配送モードについて

クラウド配送モードをご利用の場合、以下の情報を収集・処理します。

### 収集する情報

| 情報                                   | 目的                             |
| -------------------------------------- | -------------------------------- |
| メールアドレス                         | アカウント認証（Amazon Cognito） |
| Webhook URL                            | 通知の配送先として保存           |
| 通知トークン                           | 通知の認証に使用                 |
| 通知内容（タイトル・メッセージ・種別） | Webhook への転送                 |

### 収集しない情報

- ゲームのプレイ内容・セーブデータ
- 位置情報・デバイス情報

---

## 利用する第三者サービス

### Amazon Web Services (AWS)

アカウント認証（Amazon Cognito）、データ保存（DynamoDB）、API ホスティング（API Gateway / Lambda）に AWS を利用しています。

[AWS プライバシーポリシー](https://aws.amazon.com/jp/privacy/)

---

## データの保存期間

- アカウント情報・設定：退会（アカウント削除）まで保持

---

## ログアウト時の動作

ログアウト時、クラウド上にスケジュールされた通知予定はすべてキャンセルされます。
